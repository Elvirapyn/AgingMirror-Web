使用说明:

1. 初始化(当文件夹in中有变动的时候要进行初始化): 打开final,运行align.py
( 这里需要配置dlib库,最好上网百度一下怎么安装
好像需求cmake和boost库啥的 )

2. 进行比对: 运行Final.py
向Final.py里只需传入目标图片url
( 这里要安装TensorFlow,opencv,PIL啥的,要是还有报错就开console看看 )
Final.py将会反馈给你三个东东,按顺序分别是:
库中与目标图片最接近的图片url, 这俩图片的相似度, 以及该图片所属的文件夹名
